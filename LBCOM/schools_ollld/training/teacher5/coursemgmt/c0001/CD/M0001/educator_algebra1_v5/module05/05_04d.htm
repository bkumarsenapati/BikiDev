05.04 Special Lines Assignment

  1. What is the slope of the line represented by the equation y = 7?

  2. What is the slope of the line represented by the equation x = -9?

  1. Which is the equation of the red line?
    a) y = 9 b) y = -9 c) x = 9 d) x = 0

  2. Which is the equation of the blue line?
    a) y = 1 b) y = -1 c) x = -1 d) x = 0

  1. Write the equation of a horizontal line that crosses the y axis at -4.

  2. Write the equation of a vertical line that crosses the x axis at 3.

  3. Write an equation of a line with a slope of 0 and passing through the point (5,4). Hint: To answer this question, consider the type of line that has a slope of 0. Then write an equation of that line using the values in the ordered pair.

  4. Write an equation of a line with an undefined slope and passing through the point (-2, 4). Hint: To answer this question, consider the type of line that has an undefined slope. Then write an equation of that line using the values in the ordered pair.

  5. What is the slope of a line that is parallel to the line y = 4 x +5?

  6. What is the slope of a line that is perpendicular to the line y = -5x + 4?

  7. Write the equation of a line parallel to y = 4x - 5 and whose y intercept is 3.

  8. Write the equation of a line perpendicular to y = 5x + 3 and whose y intercept is 4.
