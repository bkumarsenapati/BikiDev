Text Versions

For anything that requires a version besides native interactives and images, such as interactive elements, native interactives, eLL interactives, and videos. Simply wrap the div in the following code. The text version file should be saved to "textversion" folder in the module and should only contain the content with html markup (no html, head or body tags). Examples can be seen throughout the cheatsheet.

Wrapping Div:

<div class="text_version" data-src="textversion/pagename_txta.htm">
    Interactive content div here.
</div>

Example Text Version File:

<p>This is the text version for this interactive.</p>
<p>I can have any content in here.</p>

One-Off Interactive

For interactives that are inside the interactive folder that will be loaded into an iframe as a one-off or custom interactive

This is the text version. 
<div class="interactive" data-src="interactive/GriddedResponse/index.htm:680:555">
    This is the text version.
</div>
Optional:
"notext" after the width to hide the text version button
data-src="link/to/item/index.htm:width:height:notext
