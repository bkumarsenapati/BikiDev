07.06 Trinomials Part 2 Practice Solutions

1. 4x2 + 25x + 6 = 4x2 +24x + 1x + 6

                            = 4x(x + 6) + 1(x + 6)

                            = (x + 6)(4x + 1)

4. 18x2 - 21x - 15 = 3[6x2 - 7x -5]        (GCF)

                    = 3[6x2 +3x - 10x - 5]

                    = 3[ 3x(2x + 1) - 5(2x + 1)]

                    = 3(2x + 1)(3x - 5)

2. 3x2 + 2x - 1 = 3x2 + 3x - 1x - 1

                         = 3x(x + 1) -1(x + 1)

                         = (x + 1)(3x - 1)

5. 3x2 + 12x + 7 = Prime

3. 9x2 - 16 = (3x + 4)(3x - 4)

This is a difference of squares. It is not a trinomial at all. Note, you can treat this like a trinomial if you write the missing middle term as +0x.

6. 10x2 + 2x - 8 = 2[5x2 + x - 4]         (GCF)

               = 2[5x2 +5x - 4x - 4]

                = 2[5x(x + 1) -4(x + 1)]

                = 2(x + 1)(5x - 4)
