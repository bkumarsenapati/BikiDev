03.11 Practice Test

Review for the test

At the end of Module 3, you should be able to

  • classify numbers into the real number system categories
  • add, subtract, multiply and divide integers
  • use the order of operations to solve problems
  • translate sentences into algebra equations and vice versa
  • solve problems involving absolute value
  • evaluate expressions involving powers and roots
Visit the review page for Module 3. This will help you prepare for the final assessment.

This test is a practice test for the final assessment of the module.

After you have completed all the assignments in the module, you should take this practice test. If your score is below 75% on the practice test, please contact your instructor. He/she will be able to give you some study tips in order to improve your grade on the final module assessment.
