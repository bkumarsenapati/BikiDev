05.11 Solving Systems of Equations

The Addition Method for solving systems of equations

Graphing is a time consuming process and why not use a short cut if one exists?
The second method for solving systems of equations is called the Addition method.

The objective of the Addition method is to create two equations so that when you add them together, one of the variables will add out. Here is what that means:

Let's solve the following system of equations:

2x - y = 16
x + y = 5

Notice that the y terms have opposite coefficients. One of the y's is positive and the other y is negative. So when you add the y terms together, you will get zero. They will add out. This is great news because it allows us to solve for the x variable.


Here is the work. We are going to add the equations together.

Notice that the y term is gone and we can solve for x by dividing both sides of the equation by 3. This is the main objective of the Addition method. If one of the variables does not add out, then you did something wrong.

So we have the value of x but we need the value of y in order to have the point at which the lines would intersect. Remember we are looking for an ordered pair solution.

To find the value of y, substitute your x value into one of the original equations. Let's use 2x - y = 16. We know that x = 7 so here is the work for finding the value of y.

2x - y = 16
2(7) - y = 16
14 - y = 16
-y = 16 - 14
-y = 2
y = -2


The solution to this system of equations is the point (7, 2).

This is the point at which the two lines would intersect. This point will work in both equations. Substitute (7, -2) in both equations and see that the solution sets make the equations true.
