04.07 Absolute Value Equations

© 2004 clipart.com
Solving equations with absolute values is like tennis. You must have two players to make it work.

There must be two answers to an absolute value equation (unless there is no solution).
Below are some examples, and then some problems to work.

You will save time in the long run if you work the examples yourself before trying the problems.

The Concept
Explanation
Examples

|x| = 5

In the absolute value equation shown above, there are two possible values for x. If we substitute a positive 5 in the place of the x, we will get a true equation.

|5| = 5 is TRUE

If we substitute a negative 5 in the place of x, we will get a true equation.

|-5| = 5 is TRUE

This means there are two solutions to the equation.

x = 5 and x = -5

Example 1:

|x| = 6
x = 6 and x = -6

Example 2:

|x| = 17
x = 17 and x = -17

Example 3:

What if the |x| equals a negative number?
|x| = -12

What values of x would make the statement true?
|12| = -12 is FALSE

|-12| = -12 is FALSE

Since both equations are FALSE, there are "no solutions."


Basic Absolute Value Equations
Explanation
Examples

| x + 2 | = 5

In the absolute value equation shown above, there are still two possible numbers that would work in the absolute value.

To find the two values of x, write two equations and solve them. In this equation, x + 2 = 5 and x + 2 = -5. Remember the absolute value bars mean you take the positive of what is inside.

x + 2 = 5 and x + 2 = -5
x = 5 - 2 and x = -5 - 2
x = 3 and x = -7

Example 1:

|x - 4| = 12
Write two equations and solve for x.
x - 4 = 12 and x - 4 = -12
x = 16 and x = -8

Example 2:

|5x| = 15
Write two equations and solve for x.
5x = 15 and 5x = -15
x = 3 and x = -3

Example 3:

|2x + 1| = 9
Write two equations and solve for x.
2x + 1 = 9 and 2x + 1 = -9
2x = 8 and 2x = -10
x = 4 and x = -5


Look Out

The next set of examples will illustrate a necessary step for getting ALL the solutions correct. Watch and read carefully.


Advanced Absolute Value Equations

The key to solving all absolute value equations is that you must isolate the absolute value before you write the two equations.

Example 1:

|x + 4| - 5 = 10

Isolate

|x + 4| - 5 + 5 = 10 + 5
|x + 4| = 15

Write Two Equations

x + 4 = 15 and x + 4 = -15
x = 15 - 4 and x = -15 - 4
