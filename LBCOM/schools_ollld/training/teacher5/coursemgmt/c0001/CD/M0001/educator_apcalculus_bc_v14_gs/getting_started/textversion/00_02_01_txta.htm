Navigation Interactive—Text Version

Home and Arrows

Using the home icon will be the fastest route back to the course home page. The left arrow navigates to the previous page and the right arrow navigates to the next page.

Module Progress Bar

Using the module progress bar will help you to see how much of the lesson you have completed already.

You can also navigate to other pages within the lesson from the module progress bar.

Bookmark

Using the bookmark will allow you to save your place in the course so you can easily return to it later.

Glossary

Using the Glossary button will allow you to review all glossary terms used in the course.

Menu

Using the menu on the left allows you to navigate between modules.

Notes

Using the Notes button will demonstrate a system to help you organize what you read so you can easily review and prepare for tests.

Print Page

Using the Print Page button allows you to print the lesson page.
