Functions and Relations

The white graph is the graph of a relation that is also a function; the red graph is the graph of a relation that is not a function.




The vertical line test is the quickest way to tell if a graph represents a function or a relation. If a vertical line can cut the graph in only one point, then the graph is a function. If a vertical line touches the graph in more than one point, then the graph is a relation, but not a function.

Note: All functions are also relations. A function is a special relation. 		Composite functions :
