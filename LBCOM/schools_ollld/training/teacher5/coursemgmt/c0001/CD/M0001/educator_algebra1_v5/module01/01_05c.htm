01.05 Submitting Travelogue Assignments

Below is a typical assignment. You should have no problem with these questions because they come from pre-algebra. Let's see how well you do. If you succeed, please do each assignment as carefully as you have done this one.

There are three objectives for this lesson. They are:

  1. to create a folder on your computer for all of your Algebra work,
  2. to show the travel agent that you understand how learning in an online algebra class requires close reading, and
  3. to learn how to submit an assignment successfully.

You're Up!

Following are five questions that you will need to answer.
  1. -3 * -4 =
  2. 1/2 * 1/4 =
  3. Write in simplest form: 12/144
  4. Write the number 6 as a fraction.
  5. 52 =

01.05: Submitting Travelogues

Following are your steps for this first assignment. Be sure to read them all before you submit anything.

  1. Create an Algebra folder on your computer's hard drive. If you do not know how to create new folders on your computer, please read the Creating New Folders instructions.

    If you are a lab student, and you will be saving your files to disk, you will not need to do this part. Lab students are encouraged to use the My files icon on the navigation bar to the left. The My files area is a place to save your files. This will keep you from having to carry a disk around because you can access the files online.
  2. You will need to answer the five questions from above in a word processing document. You will need to have saved this as a .rtf (rich text file) in your Algebra folder created in Step 1. For help with creating a .rtf file, please read the Saving Your Work as a Rich Text File instructions. Please copy and paste the work and attach your file when you submit "01.05 Submitting."
  3. You will be submitting your answers to the five questions here. Each question is worth 2 points.

This assignment is worth 10 points.
