Audio Recording

This course uses voice recordings, so you will need some type of software or program to make an audio recording to submit to your instructor. There are several free audio options that will work for this purpose. Be sure to contact your instructor to determine the file type in which you need to submit your recordings.

  • Audacity®
    • Downloads available for Windows and Mac platforms
    • Export as .WAV, .MP3, or .Ogg
    • Documentation and tutorials
  • AudioPal
    • No download required
    • Website link to audio is provided, which will work across all platforms
    • Instructions
  • Freecorder
    • Available for Windows only
    • Runs inside your browser (Chrome, Internet Explorer, or Firefox)
    • Freecorder 6 User Guide
  • Garage Band
    • Mac only
    • Instructions from eHow.com
