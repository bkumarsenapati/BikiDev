Saving Your Work as Rich Text Files

  • Open your word processing program.
  • Work the problems on the new document; remember to show ALL of the steps as your example problems in the Practice Walk showed you to do.
  • Save the document in your Math 7 folder (you should have already created this separate folder in your word processing files); name the file as instructed on the assignment page. (ex. 303.rtf)
  • Before pressing Save, go to "Save as type:" just below the file name and click on the drop down arrow (Save as type: might also be listed as File type:, but the same process can still be followed).
  • After selecting the drop down arrow, scroll down to Rich Text Format (*.rtf) and click on it (Rich Text Format is the generic word processing file that almost all word processing programs can exchange and read).
  • Go to your assessment area and click on the correct assessment name. (ex. 3.03 Scientific Notation and the Powers of 10).
  • Attach the file to the assessment using the browse button; this is the same process that you use when you attach files to emails.
