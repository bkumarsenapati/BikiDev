Real numbers

 

	The combination of all the rational and irrational numbers is called the set 

of real numbers or the real number system. It includes all numbers that can be represented

on a number line, including such values as Π and √2. The set of real numbers extends

to infinity in both directions on a number line. The set of real numbers is typically denoted by R.


Rational number

 

	A rational number is any number that can be expressed as the quotient of two integers

in the form of a / b, where b ≠ 0. In general, if a and b are both rational numbers, then so are

a + b,  a - b,  a x b and a / b, if b is not zero. The set of rational numbers includes the set of 

whole numbers and all fractions, both positive and negative. 



Examples: 

5, -7, 0, 3/7, -5/2, -2.77` and 3.628 are all rational numbers. 

They can be written as the following ratios: 5/1, -7/1, 0/1, 3/7, -5/2, -25/9,  3628/1000.


Irrational numbers

 

	An irrational number is any real number that cannot be expressed as the ratio of two

integers. In other words, an irrational number is not a rational number. The decimal component of

irrational numbers never repeat and never terminate. 



Examples: 

√2 and Π are irrational numbers.


Integers

 

	The set of integers, usually denoted Z, is comprised of three subsets: 



- Positive whole numbers: {+1, +2, +3, +4, ...}

- Negative whole numbers: {-1, -2, -3, -4, ...}

- Zero: {0}.



Therefore, the set of integers is defined as all positive and negative whole numbers and zero.

The set of integers is an infinite set. In addition, an integer with no sign is assumed to be positive.


Whole numbers

 

	The whole numbers are the set of numbers which start with zero and are formed by successively

adding one. 



Example: 

{0, 1, 2, 3, 4, ...}


Counting numbers

 

	The counting numbers, also known as the natural numbers, start at one and increase one

at a time to infinity. 



Example: 

The counting numbers are 1, 2, 3, 4, 5, 6, ... . 



The three dots, or ellipsis, indicate that the sequence of numbers has no end.

No matter how large a number can be thought of it can always be made greater by adding 1 to it.

a, b, and c, a(b + c) = ab + ac and (b + c)a = ba + ca, and a(b - c) = ab - ac and (b - c)a = ba - bc.

This illustrates that the distributive property holds for multiplication over addition. 



Similarly, it can be illustrated that the distributive property holds for the operation of multiplication

Commutative Property

on makes no difference in the result. It is also sometimes referred to as the order property. 



Addition and multiplication of real or complex numbers is commutative. For addition, a + b = b + a, 

for any real numbers a and b. For multiplication, a x b = b x a, for any real numbers a and b.

Associative Property

 

	Also referred to as the grouping property, the associative property states that the sum, or product,

of three numbers does not depend on the way the numbers are grouped when they are added or multiplied. That is,

for all numbers a, b, and c, (a + b) + c = a + (b + c) and (a x b) x c = a x (b x c). 

Addition and multiplication are associative.

numbers does depend on the way that they are grouped when they are subtracted or divided.

Symmetric property of equality

 

	The symmetric property of equality states that if a = b, then b = a, for all real 

numbers a and b. 



Example: 

If 3 = 2 + 1, then 2 + 1 = 3.


Reflexive property of equality

 

	The reflexive property of equality states that something is always equal to itself. 

For example, a number is equal to itself. In general terms, the reflexive property of equality

states that for any real number x, x = x. 



Examples: 

3 = 3 

-5 = -5

34 = 34


Transitive property of equality

 

	The transitive property of equality states that for all real numbers a, b, and c,

if a = b and b = c, then a = c. 



Example: 

If 5 = 3 + 2 and 3 + 2 = 4 + 1, then it follows that 5 = 4 + 1.


Identity property of multiplication

 

	This property states that d is the identity element for a given set under 

multiplication if d x n = n and n x d = n, for all values of n in the set. 



Example: 

1 is the identity for multiplication for the real numbers.

1 x 5 = 5 and 5 x 1 = 5.


Identity property of addition

 

	This property states that d is the identity element for a given set under addition if 

d + n = n and n + d = n, for all values of n in the set. 



Example:

0 is the identity for addition for the real numbers.

0 + 7 = 7 and 7 + 0 = 7.


Additive Inverse

 

	The additive inverse of any real number x is its opposite, -x. The sum of 

any real number and its additive inverse is zero. If x is any real number, then x + (-x) = 0. 



Examples: 

The additive inverse of 4 is -4. 4 + (-4) = 0 

The additive inverse of -4 is -(-4) = 4. (-4) + 4 = 0.


Multiplicative Inverse

 

	The multiplicative inverse of an element, a, is the element a-1, where a-1

has the property that: a-1 x a = a x a-1 = I, where I is the multiplicative identity.

For the real numbers, the multiplicative inverse of a number is its reciprocal. The real numbers a and 1/a

are reciprocals. 



Example: 

The multiplicative inverse of 3 is 1/3 because 1/3 x 3 = 3 x 1/3 = 1.


Coefficient

 

	The number that precedes a variable or variables in an algebraic term is 

called a coefficient. 



Examples: 

The coefficient in 5ab is 5. 

The coefficient in -2x2 is -2.


Constant

 

	A constant is a term in an algebraic expression which does not contain variables. 



Example: 

14, -2.013, 53 and √2  are all constants. 



Sometimes a constant is represented by a symbol. For example, in the expression ax2 + bx + c,

c represents a constant. The symbol Π represents the ratio of the circumference to its diameter.


Term

 

	A term is a part of a mathematical expression separated by addition or subtraction signs.

It can be a number, a variable, a product , or a quotient in an expression. 



Example: 

3x + y - 4 is a three term algebraic expression with the terms: 3x, y, and 4.


Like Terms

 

	Terms that are equivalent in both the variable and the power of the variable 

are called like terms. The coefficients of the terms do not have to be the same. 



Examples: 

2x and 3x are like terms.

4y2 and 9y2 are like terms.

6ab and -2ab are like terms.

2x and 5x2 are not like terms.

7ab and -9ac are not like terms. The variable parts are different.


Expression

 

	A mathematical expression is a collection of numbers and/or variables often

including symbolic representations of any operations performed. 



Examples: 

a3 + 7x2

5 1/2 + log 3 

are both mathematical expressions.


Combining like terms

 

	Combining like terms refers to combining terms that have the same variable 

or variable powers. To combine like terms, combine their numerical coefficients using the

rules to combine signed numbers and keep the variable or variable powers. 3x and 5x

are like terms because they have the same variable x. 



Example: 

3x + 5x = 8x 

-7x2 + 5x2 = -2x2


Direct variation

 

	The relationship which exists between two variables when a change in one variable

causes a proportional change in the same direction in the other variable is called direct 

variation. One variable is said to be directly proportional to the other. 



Direct variation is usually written as y = kx, where the constant, k, is called the constant

of variation. Direct variation is a linear function passing through the origin. 



Example: 

At a constant speed, distance traveled varies directly with time. 

At 50 mph, 100 miles will be traveled in 2 hours. If the time is doubled to 4 hours,

the distance is doubled to 200 miles.


Inverse variation

 

	An inverse variation is a relation between two variables, x and y, of the general form 

y = kx , where k is said to be the constant of variation. As the value of y decreases, the value of x 

increases, and vice versa. For this reason, x and y are said to vary inversely and are inversely

proportional.


Vertical line test for a function

 

	The vertical line test for a function involves running a vertical line through the 

graph of a relation to determine if it is a function. If a vertical line crosses the graph more than

once, then the graph does not represent a function. If every vertical line crosses the graph at most

once then the graph represents a function. 



Examples:

Every vertical line crosses the graph of x + y = 1 exactly once. Therefore, x + y = 1 describes 

a function.

Therefore, x2 + y2 = 9 does not describe a function.


Linear equation

 

	A linear equation is an algebraic representation of a linear relation. It is written

as a polynomial equation of the first degree. Linear equations are usually expressed in standard form

as ax + by = c, or in slope-intercept form as y = mx + b. The graph of a linear equation is a straight line. 



Examples: 

5x + 7y = 9

y = 8x - 5


Slope-intercept form of an equation of a line

 

	An equation of the form y = mx + b, where m represents the slope of the line and b

represents the y-intercept of the line, is called the slope-intercept form of the linear equation. 



Examples: 

y = 2x + 7

y = -6x + 1 

are in slope-intercept form. 



When an equation is expressed in slope-intercept form, it can easily be graphed. 



Example: 

Consider the equation y = 2/3x - 4 



First recognize -4 as the y-intercept and the slope as m = rise/run = 2/3 .


Standard form of a linear equation

 

	A linear equation expressed in the form ax + by + c = 0, is considered to be

in standard form. 



Example: 

4x + 2y + 6 = 0


Slope of a line

 

	The slope of a line is the measure of its inclination or steepness. The grade of a

road is a measure of its steepness or slope. The pitch of a roof is a measure of the slope of the roof. 



Coordinate geometry is used to measure the slope of lines. The slope of a line is the ratio 

of the change in the vertical direction, the rise, divided by the change in the horizontal direction,

the run, necessary to get from one point on the line to another point on the line. It is defined

as follows: 



slope = (rise / run) = (change in y-coordinates / change in x-coordinates) . 


























Notice as you drag AB, the slope changes. The slope of the line fits one of these four categories: 

1. If the line rises from the left to the right, the slope is positive. 

2. If the line falls from the left to the right, the slope is negative. 

3. If the line is horizontal, the slope is zero. 

4. If the line is vertical, the slope is undefined.


X-intercept

 

	The x-intercept gives the position of a point, or points, where the graph of a function

crosses, or intercepts, the x-axis.


Y-intercept

 

	The y-intercept gives the position of the point, or points, where the graph of a

function crosses, or intercepts, the y-axis.


Point-slope form of an equation of a line

 

	An equation of the form y - y1 = m(x - x1) for the line passing through a point 

whose coordinates are (x1, y1) and having a slope of m is said to be the point-slope form of 

an equation of a line. If a particular point (x1, y1) on the line is known, the slope between

any pair of points is:



m = (Dy / Dx) = (y - y1 / x - x1).



This simplifies to give the point-slope form:

y - y1 = m(x - x1).


Parallel Lines

 

	In Euclidean geometry, parallel lines are lines which maintain the same distance

apart and therefore never meet. They have the same direction and their slopes are equal. So, 

if m1 and m2 are the slopes of lines AB and CD respectively, and m1 = m2, then AB || CD.  

The symbol || is used to represent two parallel lines. 

 

Example: In the following figure, AB || CD. 








The two lines are marked with the same symbol to indicate that they are parallel.

A line and a plane are parallel if they do not intersect. Two planes are parallel 

if they do not intersect. 








If a line intersects a pair of parallel lines a number of relationships are created

among the angles that are formed. The intersecting line is called a transversal. 








The relationships are:


  •  Alternate interior angles are congruent so m∠3 = m∠6 and m∠4 = m∠5.


  •  Alternate exterior angles are congruent so m∠1 = m∠8 and m∠2 = m∠7.


  •  Corresponding angles are congruent so m∠3 = m∠7 and m∠1 = m∠5; m∠4 = m∠8 and m∠2 = m∠6.


  •  Interior angles on the same side of the transversal are supplementary so, m∠3 + m∠5 = 180° 

 and m∠4 + m∠6 = 180°.

    Coordinate System

     

    	A pair of number lines that are drawn perpendicular to each other through their 

zeros describes a coordinate system. The horizontal number line is called the x-axis

and the vertical number line is called the y-axis. Their point of intersection is called 

the origin. The axes divide the plane into four regions called quadrants, numbered 

counterclockwise starting in the upper right, I, II, III, and  IV. This makes it possible

to name the location of any point in the plane by specifying its coordinates in the form

(x, y). Coordinates in the form (x, y) are called ordered pairs.



The coordinate system is referred to as the Cartesian coordinate system in honor of the 

French mathematician and philosopher Rene Descartes, who showed how algebra and geometry 

could be related.


    X-axis

     

    	The x-axis in a coordinate system is the horizontal axis. It is perpendicular

to the y-axis. Its numbering starts with 0 at the origin and goes to infinity to the right

of the y-axis and negative infinity to the left of the y-axis.


    Y-axis

     

    	The y-axis in a coordinate system is the vertical axis. It is perpendicular

to the x-axis. Its numbering starts with 0 at the origin and goes to infinity above the 

x-axis and negative infinity below the x-axis.


    Quadrant

     

    	The two perpendicular number lines that make up the Cartesian coordinate system

divide the plane into four sections, or quadrants. The quadrants are labeled as shown, 

using Roman numerals.  



Quadrant	Coordinate

I		(x, y)

II		(-x, y)

III		(-x, -y)

IV		(x, -y)


    Ordered Pair

     

    	An ordered pair is a pair of numbers which occur in a designated order.

When plotting points on the Cartesian coordinate system ordered pairs in the form (x, y) 

represent points on the coordinate plane. The first element, the x-coordinate, represents the 

horizontal distance between the origin and the point.  The second element, the y-coordinate,

represents the vertical distance between the origin and the point.


    Abscissa

     

    	The x-coordinate of a point on a Cartesian coordinate system is called 

the abscissa of the point. The abscissa is the horizontal position of the point 

in relation to the origin (0, 0).


    Ordinate

     

    	The ordinate is the name given to the y-coordinate of an ordered pair on

the Cartesian coordinate plane. The ordinate is the vertical position of the point 

in relation to the origin (0, 0).


    Polynomial

     

    	A polynomial is an algebraic expression with one or more terms. Terms are separated 

by plus or minus signs. The coefficients of the terms must be real numbers and the exponents must 

be nonnegative integers. Polynomials are classified according to the number of terms they have. 



Examples: 

6x	 			1 term, a monomial.

3x + 2 			2 terms, a binomial.

4x2 + 3x - 2 		3 terms, a trinomial.


    Degree of a polynomial

     

    	The general form of a polynomial can be expressed as axn + bxn-1 + cxn-2 + .... + k, 

where n is a nonnegative integer and a, b, c, ..., k are coefficients that stand for any 

real numbers and a ≠ 0. The highest power of the variable x indicates the degree of 

the polynomial. This power must be a positive number.

If a polynomial has two or more variables, the degree is equal to the greatest sum of 

the exponents in any term. The degree of the term x3y2 is 5.


    Monomial

     

    	An algebraic expression consisting of one term. The term can be a constant,

a variable, or the product of a numerical value and one or more variables. 



Examples: 

5 or -5 			a constant such as 5 or -5

x4 		a variable raised to a power

8x2 		the product of a number and a variable raised to a power.


    Binomial

     

    	A binomial is an algebraic expression with two terms. The terms are separated

by addition or subtraction signs. 



Examples: 

2x - 5

3xt + 1

3 + 4c

are binomials.



5x is not a binomial because it has only one term. An expression with only one term

is called a monomial. 4xy2 + 5y - 9 has three terms and is called a trinomial.


    Trinomial

     

    	A trinomial is an algebraic expression consisting of 3 terms. 



Examples: 

2x + 3y - 6

5x2 - 8x + 5


    Linear polynomial

     

    	A linear, or first degree polynomial, is a polynomial of degree 1.

It can be expressed in the form: 

p(x) = ax + b. 



Examples:

p(x) = 5x + 4

q(x) = 17x - 5


    Quadratic polynomial

     

    	A quadratic polynomial is a polynomial of degree 2. The highest power to which the variable

in a term can be raised is 2. 



Examples: 

2x2 + 10x - 7

3x2 - 2x

9x2 + 5x + 2

are quadratic polynomials.


    Parabola

     

    	A parabola is a set of points which are equidistant from a line called the

directrix and a point, the focus. The general form of the equation of a parabola is 

y = ax2 + bx + c, where a ≠ 0. The simplest parabola has 

the equation y = x2. Reflections, stretches, and translations performed

on y = x2 can generate other parabolic functions. Parabolic functions have a 

vertical line of symmetry and a turning point called a vertex. A parabola is a conic 

section generated when a cone is intersected by a plane parallel to one of 

the slanted sides of the cone.


    Maximum point

     

    	This is a point on a function where the value of y is the greatest. It is the highest point 

on a graph of the function.



Example: 

The graph below represents y = -x2 + 4x. The point A(2, 4) is a maximum point. It is 

the highest point on the graph.


    Minimum point

     

    	This is a point on a function where the value of y is the smallest. It is the lowest 

point on a graph of the function.



Example: 

The graph below represents y = x2 - 4x. The point A(2, -4) is a minimum point. It is the lowest 

point on the graph.


    Root of an equation

     

    	A solution of an equation, or a value for the variable or unknown that makes the equation true, 

is called a root of the equation. A root is said to satisfy the equation. 



Examples: 

The root of the equation x + 5 = 12 is x = 7 because the value 7 satisfies the equation.

The equation x2 + 1 = 26 has two roots, +5 and -5. Both values satisfy the equation. 



When more than one root exists, the solution is called a solution set.


    Zero of a function

     

    	This is a value which, when placed in a function, results in a value of 0 for the function. 

In general, a zero of a function f(x)is any input x such that f(x) = 0.

It is also called the root of the function. 



Examples: 

The function f(x) = x - 2 has a value of zero when x = 2, so x = 2 is a zero of f(x) = x - 2.

In the graph, the x-intercept is the zero of the function. 




    

 



The function f(x) = x2 + 2x has a value of zero when x = -2, so x = -2 is a zero,

or root, of f(x) = x2 + 2x. It also has a value of zero when x = 0, so x = 0

is a root of f(x) = x2 + 2x. In the graph, the x-intercepts are the zeros of the function.


    Solution of an equation

     

    	The values for the variables that will satisfy the equation. These values are also called the roots of the equation. 



Example: 

2x + 5 = 19 has only one solution, x = 7; 7 is the only value that satisfies the equation. 



Sometimes equations have more than one solution. 



Example: 

x2 = 1 has two possible solutions, x = 1 and x = -1; both 1 and -1 satisfy the equation.


    Scientific notation

     

    	Scientific notation is a method used to express very large or very small numbers in a more 

compact manner. The number is represented using the product of two factors; a number ≥ 1 and < 10,

and a power of ten. The first factor may contain any number of digits but the decimal point must 

always be placed so there is only one digit to the left of the decimal. Usually a maximum of four 

digits is shown after the decimal. The digits in the first factor are all significant digits. The

second factor is the appropriate power of 10. 



Examples: 

32,579 becomes 3.2579 x 104. 

0.000325 becomes 3.25 x 10-4.
