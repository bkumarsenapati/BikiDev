07.09 Quadratic Formula

Solving by factoring is the most preferred method for solving a quadratic equation, however, there are times when you will not be able to factor. If your expression is prime, you are stuck!

The quadratic formula allows you to find the values of x even when the quadratic polynomial is not factorable.

The Quadratic Formula

For the quadratic equation of the form ax2 + bx + c = 0, the value of x can be found by using the following formula.

There are a few things you need to know in order to work with the quadratic formula.

  1. The equation must be set equal to zero and should be put in descending order before trying to utilize the quadratic formula. The example below is set up properly.
    Example: 2x2 + 3x - 6 = 0
  2. The values for a, b, and c will be pulled out of the specific quadratic equation you are trying to solve.
    "a" is the coefficient for the x2 term
    "b" is the coefficient for the x term and
    "c" is the constant at the end of the equation
  3. Follow the order of operations when simplifying the quadratic formula. You must simplify the entire expression under the square root first (starting with the exponent).

WARNING: Most of the time, students make mistakes when they depend on calculators to simplify expressions like these for them. You must be aware of how the calculator interprets what you type in. In most cases it is easier to do the arithmetic in your head and only use the calculator when the numbers are large.

For example: (-3)2 = 9, but if you type -32 into your calculator, you will get the answer -9. This will make your final answer for the quadratic formula incorrect.

Wouldn't it be easier to do (-3)2 in your head? You know the answer should be +9.


Example 1:

2x2 + 3x - 6 = 0

Remember that factoring is the simplest method for solving quadratics BUT this polynomial is prime so factoring would not work. We will use the quadratic formula instead.

      2x2 + 3x - 6 = 0

a = 2     b = 3    c = -6

 Step 1: Identify the values of a, b and c from the given equation.

Step 2: Substitute the values of a, b and c into the formula and simplify the expression under the radical.

(3)2 = 9

-4(2)(-6) = +48

Note: Since the square root of 57 is not a whole number, simply leave the 57 under the radical for your answer.

2x2 + 3x - 6 = 0

 Step 3: Write the 2 solutions for x. (Sometimes there may only be 1 solution.)
You can use a calculator to approximate these values if necessary. Real life problems involving distance or determining the x intercepts may require you to do this. For our purposes, you can leave your answers in this simplified radical form.

Example 2:

x2 - 4x - 3= 0

Remember that factoring is the simplest method BUT this polynomial is prime so factoring would not work. We will use the quadratic formula.

        x2 - 4x - 3= 0

a = 1    b = -4     c = -3

 Step 1: Identify the values of a, b and c from the given equation.

Step 2: Substitute the values of a, b and c into the formula and simplify the expression under the radical.

(-4)2 = +16

-4(1)(-3) = +12

Note: Since the square root of 28 is not a whole number, simply leave the 28 under the radical for your answer.

Later in the course, we will learn how to simplify this type of radical.
Step 3: Write the 2 solutions for x. (Sometimes there may only be 1 solution.)
Again, you can use a calculator to approximate these values if necessary. Real life problems involving distance or determining the x intercepts may require you to do this. For our purposes, you can leave your answers in this simplified radical form.

Example 3:

2x2 + 5x = 3

First we must write this equation so that it is equal to zero and so the terms are in descending order.

2x2 + 5x - 3 = 0

Now, this equation does factor, but let's solve it using the quadratic formula anyway.

       2x2 + 5x - 3 = 0

a = 2       b = 5     c = -3

Step 1: Identify the values of a, b and c from the given equation.

Step 2: Substitute the values of a, b and c into the formula and simplify the expression under the radical.

(5)2 = +25

-4(2)(-3) = +24

Note: Since the value under the radical is a perfect square, you can simplify the sqrt of 49.

So, x = 1/2 or x = -3.

Example 4:

x2 - 2x + 1 = 0

This equation does factor, but let's solve it using the quadratic formula anyway.

       x2 - 2x + 1 = 0

a = 1        b = -2      c = 1

Step 1: Identify the values of a, b and c from the given equation.

Step 2: Substitute the values of a, b and c into the formula and simplify the expression under the radical.

(-2)2 = +4

-4(1)(1) = -4

Note: Since the value under the radical is zero, the square root of zero is simply zero.

So, x = 1.

Step 3: Write the solution for x.

Since there is no difference between adding 0 or subtracting 0, there will only be one answer to this equation.

If we would have factored this quadratic equation, we would have had (x - 1)(x - 1) = 0. Solved for x, our solution would be x = 1.
