Solutions - Solving Pure Quadratics

Part One

2 and 3 because they have a squared term (x2 or a2 - term), but no linear term (x or a-term).

Part Two

  1. x2 = 20
    Öx2 = Ö20
    x = Ö4(5)
    x = ± 2Ö5

  2. 81x2 - 49 = 0
    81x2 = 49
    x2 = 49/81
    Öx2 = Ö49/81
    x = ± 7/9

  3. 9a2 + 16 = 0
    9a2 = -16
    a2 = -16/9
    Öa2 = Ö-16/9


The square root of a negative number is not a real number, so there are no real solutions.
