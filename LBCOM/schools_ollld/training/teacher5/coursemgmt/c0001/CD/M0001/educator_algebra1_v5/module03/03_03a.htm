03.03 PEMDAS

Evaluating expressions

  • When you evaluate expressions, remember to use the order of operations - the order in which you evaluate expressions. You may have heard of the saying "Please Excuse My Dear Aunt Sally." This is an acronym for the order in which you must do math.

Parenthesis, Exponents, Multiply and Divide (from left to right), Add and Subtract (from left to right).

In math, there is an order to things. You must work the problems in a particular order to ensure that you get the right answer.

Following is the Order which we must follow when evaluating expressions:

Parentheses -> Exponents -> Multiplication -> Division -> Addition -> Subtraction
