07.11 Discriminant Practice Solutions

Working with the discriminant

Graphing the equation to verify the results
  1. f(x) = x2 - 3x + 8

    Using the discriminant
    b2 - 4ac where a = 1, b = -3 and c = 8

    = (-3)2 - 4(1)(8)
    = 9 - 32
    = -23

    Since the value of the discriminant is negative, there are no real roots to the equation.
  1. f(x) = 2x2 + x - 5

    Using the discriminant
    b2 - 4ac where a = 2, b = 1 and c = -5

    = (1)2 - 4(2)(-5)
    = 1 + 40
    = 41

    Since the value of the discriminant is positive, there are two real roots to the equation.
  1. f(x) = x2 - 10x + 25

    Using the discriminant
    b2 - 4ac where a = 1, b = -10 and c = 25

    = (-10)2 - 4(1)(25)
    = 100 -100
    = 0

    Since the value of the discriminant is zero, there is one real root to the equation.
  1. f(x) = x2 + 8x + 16

    Using the discriminant
    b2 - 4ac where a = 1, b = 8 and c = 16

    = (8)2 - 4(1)(16)
    = 64 - 64
    = 0

    Since the value of the discriminant is zero, there is one real root to the equation.
  1. f(x) = 3x2 + x - 2

    Using the discriminant
    b2 - 4ac where a = 3, b = 1 and c = -2

    = (1)2 - 4(3)(-2)
    = 1 + 24
    = 25

    Since the value of the discriminant is positive, there are two real roots to the equation.
