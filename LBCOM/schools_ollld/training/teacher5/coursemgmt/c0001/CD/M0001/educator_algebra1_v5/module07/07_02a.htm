07.02 Difference of Squares

Now, that you have mastered step 1 in the factoring flow chart, lets move on to step 2. The second step requires us to factor based on the number of terms in the polynomial. In this lesson, we will learn what to do when there are only two terms in the polynomial (binomial). For now, there is only one type of binomial that we will try to factor. Later, in Algebra 2, you will learn how to factor other types of binomials.

  • What is a Difference of Squares?     
  • Where did the formula come from?

These are both good questions. Let's start with the first question.

Just like its name sounds, a difference of squares is a polynomial that...

  • has two terms
  • which are subtracted, and
  • which are each perfect squares.

Perfect squares are numbers that have a square root which is an integer, for example 1, 4, 9, 16, 25, 36, etc. The exponents on variables of perfect squares are always even numbers, for example x2,x4,x6,...x2n

The formula for factoring the Difference of Squares is a2 - b2 = (a + b)(a - b).  

This formula came from the work we did in lesson 6.06, Special Products.

One of those special products was "Multiplying the Sum and Difference of the Same Terms."
It looked like this... (a + b)(a - b) = a2 - b2.

Back then we learned a shortcut for multiplying this very special case. Now, in this module, we are learning how to take the answer from the multiplication and work backwards to get the original problem. This is factoring.

 Example 1
Factor 9x2 - 4.

  9x2 - 4 = (         )(         )
(3x)2 - (2)2		 First, identify the perfect squares.

9x2 - 4 = (3x + 2)(3x - 2)

Using the formula, a2 - b2 = (a + b)(a - b), determine the values for "a" and "b" from the problem. Now, substitute them into the factored answer.

Notice how each term was written first as a perfect square? That helps us see what goes into each position in the difference of squares formula.

You can check your work by simply using the FOIL method to re-multiply the factored answer. If you arrive back at the original problem, you know that you factored correctly.

(3x + 2)(3x - 2) = 9x2 - 6x + 6x - 4
                          =9x2 - 4 check


 Example 2
Factor x2 - 36y4.

   x2 - 36y4 = (         )(         )
(x)2 - (6y2)2 		First, identify the perfect squares.

x2 - 36y4  = (x - 6y2)(x + 6y2)

 Using the formula, a2 - b2 = (a + b)(a - b), determine the values for "a" and "b" from the problem. Now, substitute them into the factored answer.

Again, you can check your answer, but re-multiplying the two parentheses.

 Example 2
Factor 2x2y2 - 50.

At first it doesn't seem as though this is a difference of two squares. It does have two terms that are subtracted but the terms are not perfect squares. However, there is a GCF to be removed so we need to do that and then look again.

2x2y2 - 50 = 2(x2y2 - 25)

Now we can see the difference of two squares in the parentheses.

2(x2y2 - 25)  = (         )(         )
  (xy)2 - (5)2		 First, identify the perfect squares.
2(x2y2 - 25)  = 2(xy + 5)(xy - 5) Using the formula, a2 - b2 = (a + b)(a - b), determine the values for "a" and "b" from the problem. Now, substitute them into the factored answer.

Notice how the GCF, 2, was kept in the problem all the way to the answer?

If you have a GCF in a problem, you must factor it out first and make sure it is part of the final answer.
