Collaborative Lesson Assigned Problems

Give an example for five of the following series. You may use examples from your course but may not include problems that you have submitted for graded homework. You may use a series more than once if more than one test applies. Where appropriate include your work using the specified test.

  1. Divergent series using the nth term test

  2. Convergent geometric series. Include the computation of the sum.

  3. Convergent p-series

  4. Divergent p-series

  5. Convergent series using the integral test. Include the maximum remainder if the partial sum is found using the 6th term.

  6. Convergent test using the alternating series test. Include the maximum remainder if the partial sum is found using the 6th term.

  7. Convergent series using the comparison test

  8. Convergent series using the limit comparison test

  9. Convergent series using the ratio test
