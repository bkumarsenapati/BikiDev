Requirements

Hardware

  • Minimum: Pentium-II with 64MB RAM (or Equivalent)

Operating Systems

  • Windows 95, 98, 98-SE, ME
  • Windows NT 4.0, 2000 Pro, XP
  • Red Hat LINUX 7.0 or newer (Other equivalent LINUX distributions)
(MAC OS with Mozilla or Firefox should also work. [Not tested thoroughly.])

Cookies

Requires cookies to be enabled in the web-browsers.

Hints to enable cookies in various browsers:

Internet Explorer

Netscape Navigator

Mozilla

Tools -> Internet Options... -> Privacy -> Advanced ->
1) Check Override automatic cookie handling
2) Accept First Party cookies
3) Third Party cookies
4) Check Always allow session cookies

 Edit -> Preferences -> Privacy&Security -> Cookies ->
1) Check Enable all cookies

Edit -> Preferences -> Privacy&Security -> Cookies ->
1) Check Allow all cookies
2) Check Accept cookies normally

Pop-up window

Web browser settings (and any additional utility, if installed) must permit pop-up windows.


Screen Resolution and Monitor:-

A 15" monitor with a resolution of 1024 x 768 is recommended.
A 15" monitor with a resolution of 800 x 600 is the minimum requirement.
