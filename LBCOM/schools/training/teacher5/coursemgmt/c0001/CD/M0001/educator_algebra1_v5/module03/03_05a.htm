03.05 Absolute Value

Absolute value is a valuable part of evaluating expressions. Absolute value is the distance that a number is from zero. If you were asked, how far is the number -9 from zero, you would say "9 spaces." This means the absolute value of -9 is 9.

The problem is written |-9| = 9.

Absolute value is not the same as opposite. Let's consider the absolute value of a positive number. How far is the number 3 from zero? "3 spaces" is the answer. So what is a rule for absolute value?

The absolute value of a number is ALWAYS positive (with the exception of zero).

Examples
 The order of operations is still in effect. You treat the absolute value much like parentheses.

Number 1

3 - |4 - 6|
3 - |-2|
3 - 2 = 1
