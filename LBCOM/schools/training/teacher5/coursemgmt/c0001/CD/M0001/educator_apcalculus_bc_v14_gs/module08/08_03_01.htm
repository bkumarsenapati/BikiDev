Table Talk

In this lesson you will learn how to reason with tabular data. This is a recurring topic on the exam, so pay close attention!

Lesson Objectives

After completing this lesson, you will be able to:

  • use integrals to define functions
  • explore the relationship of functions defined by integrals

To meet these objectives you will review the following instructional videos:

  • Using Calculus With Discrete Data (Accumulated Change from a Rate of Change)
  • Using Calculus With Discrete Data (Accumulated Change from a Rate of Change), Cont.
