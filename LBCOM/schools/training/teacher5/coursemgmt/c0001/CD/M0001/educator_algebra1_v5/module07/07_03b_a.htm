7.03 Factoring by Grouping Practice Answers

1. ab + 7a + 2b + 14

= a(b + 7)    +2(b + 7)

= (b + 7)(a + 2)

4. 11x2 + 33x - xy - 6y

= 11x(x + 3) - y(x + 6)

Answer: prime

2. 9x2 + 3xy + 3x + y

= 3x(3x + y) + 1(3x + y)    

Note: The +1 can be written as the GCF when there is nothing else that the terms have in common.

= (3x + y)(3x + 1)

5. 2xy + 3x - 2yz - 3z

= x(2y + 3) - z(2y + 3)

Watch those signs!

= (2y + 3) (x - z)

 

3. 4ab - 24a - 5b + 30

= 4a(b - 6) - 5(b - 6)

= (b - 6)(4a - 5)

6. 6x2 - 4xy - 12x + 8y
(Don't forget to look for the GCF of the entire polynomial FIRST!)

= 2[3x2 - 2xy - 6x + 4y]

= 2[ x(3x - 2y) - 2(3x - 2y) ]

= 2 (3x - 2y)(x - 2)
