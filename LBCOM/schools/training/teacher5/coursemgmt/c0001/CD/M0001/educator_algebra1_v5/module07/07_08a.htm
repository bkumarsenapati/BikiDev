07.08 Solving Equations by Factoring

The BIG Picture

You are now entering a very important part of your Algebra World. The pieces are coming together for your understanding of Algebra as a whole. You have been learning skills that allow you to do two things. One is to solve equations and the second is to graph them.

Please read this next section carefully. It will help you organize all the things you have learned so far.

In modules 4 and 5, we learned about linear equations.

  • y = 5x + 3 is an example of a linear equation. Notice the highest power of x is x1.
  • For an equation to be linear, it may have 1 or 2 variables and must have no exponents on the variables other than 1.
  • The process of solving equations helps us find points on the graph of the linear equation.
  • The graph of a linear equations is a line.

In Module 6, you built skills to help you learn to manipulate polynomials. Now, in Module 7, you learned how to factor. These skills are needed to work with quadratic equations.

  • A quadratic equation is an equation in the form y = 2x2 + 17x + 8 . Notice the highest power of x is x2. This is how you will know you are working with a quadratic equation.
  • For an equation to be quadratic, it may have 1 or 2 variables and must have an x2 term as the highest power of x.
  • Factoring polynomials will help us solve quadratic equations. The solutions to a quadratic equation are the points where the graph crosses the x-axis. This helps us graph a parabola.
  • The graph of a quadratic equation is called a parabola. A parabola is a U shaped graph. You will learn more about graphing parabolas in future math courses.

So, how does this new lesson fit into the big picture?

Well, you have learned to factor for the sole purpose of being able to solve a quadratic equation.

Why do you need to solve a quadratic equation? To find points on the graph of the parabola. You should be beginning to see how it all fits together.

Now, it is time to jump into the new lesson. Let's take a look at solving quadratic equations.

Solutions to Quadratic Equations

A quadratic equation is of the form y = ax2 + bx + c. In the lesson and assignments in this course, you will see equations like ax2 + bx + c = 0. Basically, the y has been replaced with the number zero.

If you recall in lesson 4.06, we learned how to find the x-intercepts of a linear equation by replacing y with zero and solving for x. The same idea is applied here. When we replace y with zero, we are able to solve for x. The solutions for x will be the x-intercepts for the graph of the quadratic equation.

Quadratic equations can have 2 solutions, 1 solution or no solutions. Most of the time, quadratic equations have 2 solutions. When you look at a picture of a parabola (a U shape), you can see these 2 solutions or x-intercepts. You can also imagine that if the graph was shifted up or turned the right way, the graph would not intersect the x-axis at all.


Solving Equations by factoring

Please write these General Steps in your notes and use them to follow along with examples.

Step 1: Write the equation so that zero is on one side of the equals sign.

Step 2: Factor the expression completely. (Use your factoring chart from lesson 7.06 to ensure you factor completely and accurately.)

Step 3: Set each factor equal to zero and solve for the variable.

To make sure you understand that last step, lets take an example that is already factored and equal to zero.

(x - 4)(x + 3) = 0

Notice you have two items multiplied to equal the number zero. This is very important.

There is a property for zero that states if two terms are multiplied together to equal zero, one or both of the terms must be the number zero. If you were asked "What number times 5 equals 0?", you should know the answer. The missing number must have been zero.

We know that in our factored equation above, we have two factors multiplied to equal 0. Because of this zero property, we know that either the (x - 4) must equal zero or the (x + 3) must equal zero. So, we write each factor equal to zero and solve. Notice we have two solutions for x.

x - 4 = 0       or        x + 3 = 0

   x = 4          or          x = -3


Example 1: Solve  3x2 = 12x  by factoring.

3x2 =12x

3x2 - 12x = 0

 Step 1: Write the equation so that zero is on one side of the equals sign.
3x(x - 4) = 0

Step 2: Factor the expression on the left side of the equation completely.

3x = 0      or      x - 4 = 0

 x = 0       or        x = 4

Step 3: Set each factor equal to zero and solve for x.

There are two solutions here, x = 0 or x = 4.


Example 2: Solve  x2 - 7x + 12 = 0  by factoring.

x2 - 7x + 12 = 0

Step 1: Write the equation so that zero is on one side of the equals sign. (This step is already done in this example.)
(x - 4)(x - 3) = 0

Step 2: Factor the expression on the left side of the equation completely.

x - 4 = 0      or      x - 3 = 0

   x = 4        or        x = 3

Step 3: Set each factor equal to zero and solve for x.

There are two solutions here, x = 4 or x = 3.


Example 3: Solve  6x2 - 9x = 60  by factoring.

6x2 - 9x = 60

6x2 - 9x - 60 = 0

Step 1: Write the equation so that zero is on one side of the equals sign.

3(2x2 - 3x - 20) = 0

3(2x + 5)(x - 4) = 0

Step 2: Factor the expression on the left side of the equation completely.

(Notice the GCF was taken out first, then the trinomial was factored by grouping.)

2x + 5 = 0      or      x - 4 = 0

  

Step 3: Set each factor equal to zero and solve for x.

(The number 3 in the front of the factored expression will not be part of our solving for x process because it doesn't contain a variable.)

Sometimes students are confused by where the x = -5/2 came from. Remember you are solving the equation 2x + 5 = 0.

Example 4: Solve  x2 + 6x + 9 = 0  by factoring.

x2 + 6x + 9 = 0

Step 1: Write the equation so that zero is on one side of the equals sign. (This step is already done in this example.)
(x + 3)(x + 3) = 0

Step 2: Factor the expression on the left side of the equation completely.

x + 3 = 0      or      x + 3 = 0

   x = -3        or        x = -3

Step 3: Set each factor equal to zero and solve for x.

Because the two factors were identical, there is only one solution, x = 3.
