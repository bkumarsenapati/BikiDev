07.08 Solving Equations by Factoring Practice Answers

  1. x2 + 3x + 2 = 0

    (x + 2)(x + 1) = 0

    x + 2 = 0    or   x + 1 = 0
      x = -2      or     x = -1

  2. 9x2 - 16 = 0

    (3x + 4)(3x - 4) = 0

    3x + 4 = 0      or    3x - 4 = 0
           3x = -4    or         3x = 4
           x = -4/3  or        x = 4/3
  1. 2x2 - x = 3
    2x2 - x - 3 = 0

    (2x - 3)(x + 1) = 0

    2x - 3 = 0    or    x + 1 = 0
       x = 3/2     or     x = -1

  2. 5x2 - 10x = 0

    5x(x - 2) = 0

    5x = 0    or    x - 2 = 0
      x = 0    or        x = 2
