05.12 Systems of Equations Part 2

Great - you made it this far! Let's continue with system of equations.

We can solve systems by graphing the equations and finding the point of intersection. This gives an ordered pair (solution set) that works for both equations.

We can solve a system if the coefficients of one of the two variables are opposites (for example -4x and 4x are opposites) by adding them so one variable drops out (or adds out).

BUT - what happens if we can't use addition? For instance, look at the next two systems.

a) 25x + 16y = 91
    16x + 16y = 64

b) 2r +5s = 11
    3r - 2s = -12

In example a , one variable will not add out because the signs are not opposites.
In example b we do not have any coefficients that are the same so we can't add out a term.

In this lesson you will learn how to solve systems that require subtraction and may use multiplication.

Solving Systems of Equations Using Subtraction

If you look at a system and see that the coefficients of one variable are opposites, then use the addition method you learned in the last section.

If that is not the case, look to see if the coefficients are the same but are not opposites. The following system is an example of this.

Equation 1: 25x + 16y = 91
Equation 2: 16 x + 16y = 64

Notice that we have two terms that are identical in each equation. 16y is in both the top and bottom equation. Unfortunately, we can't add the equations together because the 16y's are not opposites and won't cancel. We can subtract the bottom equation from the top equation though! If we subtract the bottom equation from the top equations, the 16y's will in fact cancel.

Remember the rules for subtraction: Change the subtraction sign to an addition sign and take the opposites of the numbers after the sign. Follow the rules for addition.

We are now in great shape to add out the y term. Add the equations because the y terms are now opposites.

Finish solving the system. Now that you have found x, substitute x (3) into either of the equations and solve for y. Let's sub into equation 2 because it has smaller numbers and smaller numbers are easier to work with!
16x + 16y = 64
16(3) + 16y = 64
48 + 16y = 64 [add -48 to both sides]
16y = 16 [divide both sides by 16 to solve for y]

y = 1

The solution set for the system is (3, 1). This means that if you substitute x = 3 and y = 1 in both equations, the equations would be true.

Solving systems that use multiplication with the addition or subtraction

If you look at a system and see that the coefficients of one variable are opposites, then use the addition method you learned in the last section.

If that is not the case, look to see if the coefficients are the same but are not opposites. If this is the case, use the subtraction method.

BUT - what if no coefficients are the same? We can't simply add/subtract the equations and graphing takes too much time and isn't always accurate. Let's look at an example.

Equation 1: 2x + 5y = 11
Equation 2: 3x - 2y = -12		 Note that the coefficients are different

Option 1 : We can see that if we multiplied equation 1 by 3 and equation 2 by 2, the coefficients of the x terms would both be 6...that would work.

Multiply by 3------ 6x + 15 y = 33
Multiply by 2------ 6x - 4y = -24

Option 2 : We can see that if we multiplied equation 1 by 2 and equation 2 by 5, the coefficients of the y terms would both be 10...that would also work.

Multiply by 2----- 4x + 10y = 66
Multiply by 5----- 15x - 10y = -24

Would it be easier to use Option 2? Yes! We could get the y coefficients to be the same and they would already be opposites. We could just add the two equations together which is easier than subtracting the two equations!

Let's use option 2.

Now we have coefficients of y that are opposites . We can add out the y terms.



Now we will solve for y. Substitute x = -2 into either one of the original equations. We'll substitute it into both equations just to show you it really works!

The solution set for this system is (-2, 3). If you were to graph this system, at what point will the lines intersect? Try it now and you will see that the lines will intersect at (-2, 3)!

Equation1:

2x + 5 y = 11
2(-2) + 5y = 11
-4 + 5y = 11
5y = 15
y = 3

Equation 2:

3x - 2y = -12
3(-2) - 2y = -12
-6 -2y = -12
-2y = -6
y = 3
