04.12 Functional Notation

Functions can be written in other forms as well. One of the most used is called functional notation.

Look at the following equation:.
y = 2x - 3
You see a "y" and an "x" which really could be representing the x and y coordinates of a point.

Now, let's write this equation in functional notation:
f(x) = 2x - 3
What is the difference? Instead of writing a "y," we replace it with "f(x)." It looks funny but it is the same as the original equation. f(x) stands for "the function in terms of x." Writing in function notation means that you are describing the relationship between x, the input, and f(x), the output. The output is dependent upon the value of x.

Let's practice using this notation.

Example 1
What you know how to do already:
If y = 2x - 3 and x = -2, find the value of y.
Substitute -2 in for x and evaluate:
y = 2(-2) - 3
= -4 - 3
= -7.
So when x = -2, y = -7.

Now let's try it using functional notation:
f(x) = 2x - 3, find f(-2)
Notice that instead of “x” we now have “-2”. So, in the expression, wherever we see an x we substitute in -2 just as we did above.
f(-2) = 2(-2) - 3
= -4 – 3
= -7.
So again, when x = -2, y = -7.

Example 2:
f(x) = 2x - 4, find f(3)
f(3) = 2(3) - 4 = 6 - 4 = 2
so then
f(3) = 2

Example 3:
g(x) = 5(x + 1), find g(0)
g(0) = 5(0 + 1) = 5(1) = 5
so then
g(0) = 5

Composite functions are two or more functions put together.
Here is a composite function.
f(x) = -5x and g(x) = 2x - 1 Given functions f and g above, let's find f[g(3)]. Looks tough but it just is "plug and chug." Let's plug!
Starting with the inner most expression, let's find g(3).

g(3) = 2(3) - 1 = 6 - 1 = 5
Notice that we just put the value of 3 in the place of x in the g function. Since g(3) = 5, we will now find f(5)

f(5) = -5(5) = -25
Again, we put the value of 5 in the place of x in the f function.
So the answer is f[g(3)] = -25 Complete the following and put the answers in your notebook. f(x) = -2x + 1 and g(x) = 4x
