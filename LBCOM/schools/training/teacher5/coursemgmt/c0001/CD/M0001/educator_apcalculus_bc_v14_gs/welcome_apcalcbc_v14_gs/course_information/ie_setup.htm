Internet Explorer Setup

The Issue:

Internet Explorer produces a pop-up for users that are trying to access non-secure content on a secure site and vice versa. This is a problem for our courses. Depending on the choice the user selects, our content will either appear correctly or disappear completely.

Example:

Directions:

When the pop-up above appears, the user will need to click "No". If the user clicks "Yes", they will need to reset the browser and start over. You can reset the browser by clicking:

  1. Tools
  2. Internet Options
  3. Delete...
  4. Delete
  5. Ok

The User will then need to go back to the lessons area of the course where the pop-up appeared and click the "No" button when the pop-up appears.
