05.14 HONORS Word Problem Examples

Problem 1

Jovan and his family want to go snowmobiling. Winter Adventures charges $20 per day plus $.25 per mile. Snowy Expeditions charges $30 per day plus $0.15 per mile. For what mileage is the cost the same?

Solution

m = mileage
c = total cost

c = 20 + .25m
c = 30 + .15m

This problem is set up for using the substitution method of solving systems of equations. Substitute 30 + .15m for "c" in the first equation.

c = 20 + .15m
30 + .15m = 20 + .25m {Substitute 30 + .15m for c.}
30 - 20 + .15m = 20 - 20 + .25m {Subtract 20 from both sides of the equation.}
10 +.15m = .25m {Simplify.}
10 +.15m - .15m = .25m - .15m {Subtract .15m from both sides of the equation.}
10 = .10m {Simplify.}
100 = m {Solve by dividing both sides of the equation by .10.}

The cost for renting a snowmobile will be the same at 100 miles.

Problem 2

At the school store, two tablets and five pencils cost $2.25. Four tablets and four pencils cost $3.60. How much does one pencil cost?

Solution

t = the cost of a tablet
p = the cost of a pencil

2t + 5p = 2.25
4t + 4p = 3.60

This problem is set up for using the Addition Method of solving systems of equations. Multiply the first equation by -2 and leave the second equation the same. This will allow the t-terms to cancel.

-2(2t + 5p = 2.25) {Multiply entire equation by -2.}
4t + 4p = 3.60 {Leave this equation the same.}

-4t - 10p = -4.50
4t + 4p = 3.60
----------------------
-6p = -.90 { Add the equations together.}
p = .15 { Solve by dividing both sides of the equation by -6.}

Pencils cost $0.15 (fifteen cents) at the school store.

Problem 3

Two cars leave town going in opposite directions. One car is traveling 60 miles per hour (mph), and the other car is traveling 45 mph. How long will it take before the cars are 315 miles apart?

Solution

t = time
Distance = rate x time (D = rt)

distance (first car) = 60t
distance (second car) = 45t

distance (first car) + distance (second car) = total distance apart
60t + 45t = 315

60t + 45t = 315
105t = 315 { Collect like terms.}
t = 3 { Solve by dividing both sides of the equation by 105.}

In 3 hours, the cars will be 315 miles apart.

Problem 4

Two cars leave town going in the same direction. One travels 60 mph and the other travels 45 mph. How long will it take before the cars are 120 miles apart?

Solution

t = time
Distance = rate x time (D = rt)

distance (first car) = 60t
distance (second car) = 45t

distance (first car) - distance (second car) = total distance apart
60t - 45t = 120

60t - 45t = 120
15t = 120 { Collect like terms.}
t = 8 { Solve by dividing both sides of the equation by 15.}

In 8 hours, the cars will be 120 miles apart.

Problem 5

Sally paddled a canoe for 3 hours against a 5 mph current. The return trip with the current took 1 hour. Find the speed Sally was paddling in still water.

Solution

d = distance
R = rate

Distance = rate x time = time x rate

d = 3 ( r - 5)
d = 1 (r + 5)

This problem is set up for using the substitution method of solving systems of equations. Substitute 1 (r + 5) for "d" in the first equation.

D = 3 ( r - 5)
1(r + 5) = 3 (r - 5) {Substitute 1(r + 5) for d.}
1r + 5 = 3r - 15 {Use distributive property to clear parentheses.}
1r + 5 - 5= 3r - 15 - 5 {Subtract 5 from both sides of the equation.}
1r = 3r - 20 {Simplify.}
1r - 3r = 3r - 3r - 20 {Subtract 3r from both sides of the equation.}
-2r = -20 {Simplify.}
r = 10 {Solve by dividing both sides of the equation by -2.}

Sally is paddling at 10 mph in still water.

Problem 6

Roberta has a collection of dimes and quarters. She has 25 coins worth $4.45. How many quarters does Roberta have?

Solution

d = the number of dimes
q = the number of quarters

d + q = 25
.10d + .25q = 4.45

This problem is set up for using the Addition Method of solving systems of equations. Multiply the first equation by -.10 and leave the second equation the same. This will allow the d-terms to cancel.

-.10 (d + q = 25 ) {Multiply entire equation by -.10.}
.10d + .25q = 4.45 {Leave this equation the same.}

-.10d - .10q = -2.50
.10d + .25q = 4.45
---------------------------
.15q = 1.95 { Add the equations together.}
q = 13 {Solve by dividing both sides of the equation by .15.}

There are 13 quarters in Roberta's collection.

Problem 7

Five hundred fifty tickets were sold for a high school play. Adult tickets were $4.50 and student tickets were $3.00. If $2139 was collected from ticket sales, how many adults bought tickets?

Solution

a = the number of adult tickets sold
s = the number of student tickets sold

a + s = 550
4.50a + 3.00s = 2139

This problem is set up for using the Addition Method of solving systems of equations. Multiply the first equation by -3.00 and leave the second equation the same. This will allow the s-terms to cancel.

-3.00 (a + s = 550) {Multiply entire equation by -3.00.}
4.50a + 3.00s = 2139 {Leave this equation the same.}

-3.00a - 3.00s = -1650
4.50a + 3.00s = 2139
----------------------------
1.50a = 489 { Add the equations together.}
a = 326 {Solve by dividing both sides of the equation by 1.50.}

There were 326 adult tickets sold.

Problem 8

An airplane flew 4.5 hours with a 40 mph head wind. The return trip with a tail wind of the same speed took 3 hours. Find the speed of the plane in still air.

Solution

d = distance
r = rate

Distance = rate x time = time x rate

D = 4.5 ( r - 40)
D = 3 (r + 40)

This problem is set up for using the substitution method of solving systems of equations. Substitute 3 (r + 40) for "d" in the first equation.

D = 4.5 ( r - 40)
3(r + 40) = 4.5 (r - 40) {Substitute 3(r + 40) for d.}
3r + 120 = 4.5r - 180 {Use distributive property to clear parentheses.}
r + 120 - 120 = 4.5r - 180 - 120 {Subtract 120 from both sides of the equation.}
3r = 4.5r - 300 {Simplify.}
3r - 4.5r = 4.5r - 4.5r - 300 {Subtract 4.5r from both sides of the equation.}
-1.5r = -300 {Simplify.}
r = 200 {Solve by dividing both sides of the equation by -1.5.}

The speed of the plane is 200 mph in still air.
