Facts to Remember:

  1. "Times as much" means multiply.
  2. "More than" means add.
  3. "Decreased by" means subtract.
  4. "Increased by" means add.
  5. "Is, was, will be" become the equals sign (=) in algebra.

  • The set of values that can be used for the independent variable (x) is call the domain of the function. For (2, -3) and (3, -1), the domain is 2 and 3.
  • The set of values that can be used for the dependent variable (y) is called the range. For (2, -3) and (3, -1) the range is -3 and -1.

One way to express a function is to use function notation. In function notation the y or dependent variable is written in terms of x as f(x) or g(x), h(x), etc.

y = 2x+4 in function notation is f(x) = 2x+4.

To evaluate the function for a given value of x, replace x in the function with that value. When x = 4, the above function is

f(4) = 2(4)+4 =12

A composite function is two or more functions put together.

Variation and Proportion:
Direct variation means that as one variable (x) changes the other variable (y) changes in the same way. As x increases then y increases, as x decreased then y decreases. The amount of increase or decrease is called the constant of variation (k) and this is written as y = k x. These problems are usually written as proportions.

Inverse variation means that as one variable (x) increases then the other variable (y) decreases or as x decreases then y increases. This is written as y=k/x where k is the constant of variation.
