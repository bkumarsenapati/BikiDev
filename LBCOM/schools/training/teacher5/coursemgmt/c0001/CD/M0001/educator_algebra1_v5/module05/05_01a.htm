What are linear equations?

Look at equations below. Some are linear equations and some are not.

(1) 3x + 5y = 3 is a linear equation with two variables

(2) 2y = 3x - 6 is a linear equation with two variables

(3) 2a - 7 = 10 is a linear equation with one variable

(4) 4x2 + 5 = 7 is NOT a linear equation

(5) xy - 2x = y is NOT a linear equation

(6) 3/a + b/2 = 0 is NOT a linear equation

  From the examples and non examples above, figure out what makes an equation a linear equation.

Why are these equations named LINEAR Equations?

Now that you can recognize linear equations by looking at the equation, why are they called linear?

Let's examine linear equations a bit more. Look at the linear equation:

y = x + 3

Let's find some solution sets (values for x and y) that make the equation true. For example, if x = 1, substitute the value 1 for x in the equation and solve for y.

y = 1 + 3
y = 4
Therefore when x = 1, y = 4

What do you notice about the graph of the solution sets for the linear equation,
y = x+3?

If you graph the solution sets of a linear equation, the graph is a straight line.

Linear implies LINE.

A line is made of an infinite number of points. Each of these points is a solution to the equation of the line. There are an infinite number of solutions.

In equations such as y = 2x+1, y is called the dependent variable . This means the value of y depends on the value of x. If x = 2, then y = 5, if x = -1, then y = -1 and so on.

x is the independent variable because its value does not depend on any other variable in the equation.
