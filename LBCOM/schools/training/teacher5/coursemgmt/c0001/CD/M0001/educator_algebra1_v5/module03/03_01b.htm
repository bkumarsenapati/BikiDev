03.01 Number System Relationships Practice

So how can we categorize numbers using the definitions from the Lesson tab? Look at each number below and write down all the number systems to which it belongs. Roll your cursor over each number to check your answers. Remember that all the numbers that you will work with in Algebra are real numbers.


Explanations to these practice problems
  • All fractions are both real and rational. The definition of a rational number is any number that can be written as a fraction.
  • All square roots of non-perfect squares will be irrational. Irrational numbers are decimals that do not terminate or repeat. Put square root of 8 in the calculator and you will see its value.
  • Decimals that repeat or have a pattern can be written as fractions and therefore are rational.
  • The number 6 fits into all the categories. Remember that many of the number systems are subsets of each other.
