Solving Quadratic Equations by Factoring

Remember, if a quadratic equation can be factored, you are ready to solve! Don't forget about the different forms of factoring: GCF, Difference of 2 Squares, Trinomial easy and Trinomial hard. Follow the steps below whenever you want to solve equations by factoring.

Step #1: Make sure the equation is equal to zero
Step #2: Completely factor the expression
Step #3: Solve each factor equal to zero -> you must get two answers (they can be the same, if the problem is a binomial square)

Example:

Solve 3x2 =12x

Step #1: Get zero on the right side of the equation. 3x2 - 12x = 0
Step #2: Factor (GCF) -> 3x(x-4) = 0
Step #3: Solve each factor = 0
3x = 0
x = 0

x - 4 = 0
x = 4

So, the 2 solutions to the equation are x = {0,4}
Check to see if they work in the equation!

Notice how the example at the right calls the solutions "roots." They can also be written as a set of solutions.

Solving a Quadratic Equation By Factoring

Quadratic equations whose roots are integers can be solved by factoring.
Example: x2 + 4x - 21 = 0

To factor, find two numbers whose sum is 4 (the coefficient of the middle term) and whose product is -21 (the constant term). They are -3 and +7.

So, x2 + 4x -21 = (x - 3)(x + 7)

The product of two factors is 0 if one or both of the factors are 0:

x -3 = 0 x + 7 = 0
x = 3 x = -7

Check:

x2 + 4x - 21 = 0 x2 + 4x - 21 = 0
32 + 4 x 3 - 21 = 0 (-7)2 + 4 x -7 - 21 = 0
9 + 12 - 21 = 0 49 - 28 - 21 = 0
21 - 21 = 0 49 - 49 = 0
0 = 0 0 = 0
  So, the roots are 3 and -7
