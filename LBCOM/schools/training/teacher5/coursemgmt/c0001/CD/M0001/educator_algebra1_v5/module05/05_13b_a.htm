05.13 Substitution Practice Answers

1. x + 4y = 3
    2x - 3y = 17

Solve for the x in the first equation. This is best choice because it has a coefficient of 1.

x + 4y = 3
x = 3 - 4y

Substitute the value of x into the other equation.

2x - 3y = 17
2(3 - 4y) - 3y = 17

Solve for y

6 - 8y - 3y = 17
6 - 11y = 17
-11y = 17 - 6
-11y = 11
y = -1

Substitute the value of y into one of the original equations to find the value of x.

x + 4y = 3
x + 4(-1) = 3
x - 4 = 3
x = 3 + 4
x = 7

The solution to the system of equations is the point (7,-1).

2. 3x - 4y = 0
    2x + y = 33

Solve for the y in the second equation. This is best choice because it has a coefficient of 1.

2x + y = 33
y = 33 - 2x

Substitute the value of y into the other equation.

3x - 4y = 0
3x - 4(33 - 2x) = 0

Solve for x.

3x - 132 + 8x = 0
11x - 132 = 0
11x = 132
x = 12

Substitute the value of x into one of the original equations to find the value of y.

3x - 4y = 0
3(12) - 4y = 0
36 - 4y = 0
-4y = -36
y = 9

The solution to the system of equations is the point (12,9)

3.  x - y = 5
    x + 2y = 14

There are several ways to solve this problem since many of the variables have a coefficient of 1. Look at the example below to see how to solve for the x in the first equation.

x - y = 5
x = 5 + y

Substitute the value of x into the other equation.

x + 2y = 14
(5 + y) + 2y = 14

Solve for y.

5 + y + 2y = 14
5 + 3y = 14
3y = 14 - 5
3y = 9
y = 3

Substitute the value of y into one of the original equations to find the value of x.

x - y = 5
x - 3 = 5
x = 5 + 3
x = 8

The solution to the system of equations is the point (8,3).
