Using the Quadratic Formula

Solving quadratic equations by using the quadratic formula will always work! Many choose just to use this formula, but it is time consuming and it is easy to make small mistakes!

Step#1: Make sure the equation is equal to zero.
Step#2: Find a, b, and c. Be sure to include any negatives.
