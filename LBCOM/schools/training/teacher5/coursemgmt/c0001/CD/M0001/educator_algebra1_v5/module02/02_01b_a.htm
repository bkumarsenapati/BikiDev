2.01 Measure Assignment:

Making a Data Table Changed Data

 

It is recommended that you print this page for future reference.

  1. Open the program that you intend to use to make your table.
  2. There are several ways to insert a table. (If you are using a spreadsheet, you will not need to do this step because a spreadsheet is already in table form.) Somewhere on the menu across the top of your program, there will be an insert table option. You may have to look under another menu like Tools or Insert.

You may also have a shortcut button for inserting a table.

  1. After you choose to insert a table, you will be asked to select a number of rows and columns.

You will need 3 columns and 11 rows. Choose OK and the table will appear in your document.

  1. Fill in the table with the measurements that you collected from the discussion area. Here is a sample table.

  1. To save your data table, choose File, Save As from the toolbar.

  1. It is very important that you save your file both with a correct name and in the correct folder on your computer. Name and location will be important when you want to find your file and create an attachment to submit to your instructor.

Save the file in your Algebra folder that you created in Assignment 01.03. If you have not created that folder and need instructions for doing so, please click here.

Be sure to save the file in a Rich Text Format so your instructor can open it.

Now you are ready to turn in your data table to your instructor.
