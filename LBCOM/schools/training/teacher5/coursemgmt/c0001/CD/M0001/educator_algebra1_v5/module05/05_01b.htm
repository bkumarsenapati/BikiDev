05.01 Linear Equations Practice

Given the linear equation: y = 2x + 1

a) Are the following points solution sets: (0, 1), (1, 3), (-1, -1), (3, 5)
To check if these are solution sets, substitute the x and y values in the equation to see if the equation is true.
b) Graph only the points that are solutions sets to the equation.
c) Connect the points....do you get a straight line? Why?
d) Graph the point that is not a solution set for the equation. Is that point on the line? Why or why not?
