09.06 Linear Equations Review

Understanding how the following are related will help you master graphing linear equations.

  • the equation of the line
  • the table of values
  • the graph of the line
  • the solutions to the equation

After you have read these directions, work through each example in the interactivity below. It is important that you do the following.

  1. Study the equation and the table of values that go along with that equation.

  2. Choose any new value for x and find the y value that would go with it for that equation by substituting the x value in the equation and solving for y.

  3. Imagine that you are graphing the ordered pair (x, y) that you found in step 2 on the coordinate plane given. Is the point part of the red line?

  4. Check to see if the ordered pair you found is a solution to the equation. If the point is on the red line, your ordered pair is a solution to the equation. If the point is not on the red line, then your ordered pair is not a solution (check your math here because you probably made an error in solving the equation for y).

  5. Compare the graph and equation of the line to other lines and equations graphed in this section…see if you can see the differences and what caused them.

One last note: In this lesson, we will be working with equations written in slope intercept form, y = mx + b. Remember in this form, m is the slope and b represents the y-intercept (the place where the line crosses the y axis).



Before you leave this review section, it is important that you review what you have learned.
  • Any ordered pair that is on the graph of a line is a solution to that equation.
    • Example: (1, 5) is on the graph of y = 2x + 3 and it is a solution to the equation.
        • y = 2x + 3
          (5) = 2(1) + 3
          5 = 5
  • The equation of a line tells us what mathematical process is relating the value of y to the value of x.
    • Example: The line y = 2x - 3 indicates that any value for x will be multiplied by 2 and then subtracted by the number 3. The result is the value for y.
        • For the equation y = 2x - 3, if x = 2 find the value for y.
          y = 2(2) - 3
          y = 4 - 3
          y = 1
  • The value of the slope affects the steepness of the line. The larger the slope the steeper the line.
    • Example: Which line has the steeper slope?
        • y = 2x + 3

          The line y = 2x + 3 has a steeper slope.
  • A positive slope means the line rises from left to right. A negative slope means the line falls from left to right.
    • Example: Which line rises and which line falls from left to right?


        • The line with the positive 1/2 slope rises from left to right, while the line with the negative 1/2 slope falls from left to right.
  • The y-intercept value (b) determines where the line crosses the y-axis.
    • Example: What is the difference between the lines below?
        • y = 2x
          y = 2x + 3
          The lines have the same slope, but the line y = 2x + 3 is shifted up 3 spaces on the y-axis.
  • The graph of linear equations will cross the y-axis at the point where x is zero.
    • Example: Look at the table of values for the equation y = 2x + 3. What is the ordered pair for the point where the line will cross the y-axis?
        • The ordered pair (0, 3) represents the point where the line crosses the y-axis.

  • Horizontal Lines are not written in the same form as the equations above. Horizontal lines are always expressed as "y equals a number".
    • Example: y = 3
        • The graph of the line y = 3 is a horizontal line that crosses through the y-axis at the number 3.
  • Vertical Lines are not written in the same form as the equations above. Vertical lines are always expressed as "x equals a number".
    • Example: x = -2
        • The graph of the line x = -2 is a vertical line that crosses through the x-axis at the number -2.
