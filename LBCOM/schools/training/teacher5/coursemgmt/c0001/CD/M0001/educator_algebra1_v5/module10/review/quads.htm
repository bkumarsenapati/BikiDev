Quadratic equations in the form ax 2 + c = 0 are called pure quadratics. The reason for this is because the only variable in the equation is an x 2. Without the bx term, the problem is solved using square roots, because they are the opposite of squaring.

All quadratic equations can be solved by using this most famous formula: The Quadratic Formula. Unlike the other methods already mentioned, this formula works all the time! Be forewarned, this procedure can be time consuming and cumbersome at times so the other methods are preferred in many cases.
