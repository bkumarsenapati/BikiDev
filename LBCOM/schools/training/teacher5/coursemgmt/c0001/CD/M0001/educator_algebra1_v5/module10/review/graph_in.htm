Systems of Inequalities

Systems of inequalities are normally solved by graphing. The intersection of the two or three graphs is where solutions can be found. Graph each system to find the solutions.

Example #1:

Problem: Solve and graph the system of inequalities:
Equation #1 -x + y < 1
Equation #2 x + y > -3

Step #1. Solve each inequality for y as if it was an equation. Be careful though because as you're solving for y, if you have to divide by a negative number to get y by itself, you'll need to change the direction of the inequality sign. Here we don't have to worry about that.
-x + y < 1 x + y > -3
y < x + 1 y > -x - 3

Step #2. Graph each inequality as if it was an equation. The only difference is if the inequality is < or > , the line is dashed , and if the inequality is  < or > , the line is solid . (See the graph at the right, remember plot b, move m from
y = mx + b.)

Step #3. Shade the appropriate region. Since equation #1 is a "less than," the graph is shaded below first line. Since equation #2 is a "greater than," the graph is shaded above the second line. The intersection of the 2 regions is the dark gray area in the graph at the right.

Step #4. The shaded region represents all possible solutions . To check your solutions. Pick a point in the shaded region. See the point (2,0)? If x = 2 and y = 0, substitute this into each of the 2 equations.
-2 + 0 < 1 true
2 + 0 > -3 true
If you choose a point not in the shaded region, both equations would not make a "true" statement. Try it! Choose (0, 4).
