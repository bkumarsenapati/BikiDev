There are four ways to write equations of lines, depending on the information given.

  1. Write the equation from a graph by finding the y-intercept and the slope from the graph.
  2. Write an equation given the slope and the y-intercept.
  3. Write an equation given the slope and any point.
  4. Write an equation given two points.

In this example we can't use the slope-intercept formula y = mx + b because we don't know the y-intercept. So we are going to use a new formula called the point-slope formula. y - y1 = m(x - x1)

To use the point-slope formula we need to know m, x1, and y1.. In our example m = 2, x1 = -3, y1 = 4. Substituting these values in the point-slope formula
y - y1 = m(x - x1)

y - 4 = 2[x -(-3)]

y - 4 = 2(x + 3)

y - 4 = 2x + 6

y = 2x + 10

4. Write an equation of a line given two points on the
line.

Write the equation of the line that goes through the points (2,3) and (-1,6)

In the previous examples in order to write an equation of a line we have the choice of two formulas, the slope-intercept and the point-slope. Which one will we use for this example?

Since we do not know the y-intercept it looks like we should use the point-slope. But to use the point-slope formula we need to know the slope and a point. We were given two points, but we were not given the slope.

But given two points, we can find the slope of the line that goes through the two points.

Remembering from the last activity, the formula for finding slope is


y - y1 = m(x - x1 )

with m = -1 and the point (2,3) where x1 = 2, y1 = 3
y - 3 = -1(x -2)
y - 3 = -x + 2
y = -x + 5

What if we want to use the point (-1,6) instead of the point (2,3)?
That's fine; you will get the same equation. Choose the point that has smaller numbers or all positive signs, just to make it easier.
